# <a name="expected--"></a>应有“-”
您尝试创建正则表达式文本，但不包括其中一个斜杠 (/) 。 正如字符串文字在一对引号中以字符形式编写时，正则表达式文本在一对斜杠 (/) 字符中表示为个字符。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
- 插入一个终止正斜杠来标记正则表达式的结尾。
