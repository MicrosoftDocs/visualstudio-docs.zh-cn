# <a name="string-expected"></a>缺少字符串
您尝试对类型之外的对象调用 **valueOf** 方法，而不是 **调用的方法** `String` 。 此类调用的对象必须是类型 `String` 。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
- 仅对类型为的对象调用 valueOf 或 **方法。** `String`
