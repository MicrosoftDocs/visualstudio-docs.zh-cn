# <a name="the-number-of-fractional-digits-is-out-of-range"></a>小数位数超出范围
您尝试将无效参数传递给 **toExponential**。 函数 **toExponential ()** 的参数必须介于0和 20 (包含) 之间。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
- 请确保 ToExponential 的参数 **()** 不太大或太小。
