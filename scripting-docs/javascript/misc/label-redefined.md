# <a name="label-redefined"></a>标签被重定义
您创建了一个新标签，但为其赋予了现有标签的名称。 标签可用于标记代码块，但在指定范围内，它们必须是唯一的。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
- 确保程序中使用的所有标签在各自的范围内是唯一的。
