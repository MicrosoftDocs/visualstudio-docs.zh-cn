# <a name="regular-expression-object-expected"></a>应为正则表达式对象
您试图对非类型的对象调用**valueOf** **方法，而**不是调用方法。 `RegExp` 此类调用的对象必须是类型 `RegExp` 。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
- 仅对类型为的对象调用**valueOf** **或方法** `RegExp` 方法。
