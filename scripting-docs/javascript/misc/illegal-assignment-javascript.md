# <a name="illegal-assignment-javascript"></a>非法赋值 (JavaScript)
您尝试为只读标识符赋值。 不能将值分配给只读标识符。 例如，主机定义的对象和外部 COM 对象。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
- 不要为只读标识符赋值。
