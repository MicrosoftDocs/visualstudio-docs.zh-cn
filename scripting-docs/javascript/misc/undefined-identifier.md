# <a name="undefined-identifier"></a>未定义标识符
您尝试使用 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 编译器无法识别的标识符。 使用时，将返回未定义的值：  
  
- 不存在的变量，  
  
- 已声明但从未为其赋值的变量  
  
- 不存在的对象属性。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
- 用 **var** 语句声明变量， (如 `var` x; ) 中所示。
