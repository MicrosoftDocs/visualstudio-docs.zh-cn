# <a name="expected--javascript"></a>应有“(”(JavaScript)
试图将表达式括在一组括号中，但不包括左括号。 某些表达式必须包含在一组左括号和右括号内。 请注意，在以下示例中使用了圆括号。  
  
```JavaScript  
for (initialize; test; increment) {  
statement;  
}  
```  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
- 将左括号添加到计算表达式。
