# <a name="expected-"></a>应有“]”
您对数组元素进行了引用，但未包含右大括号。 引用数组元素的任何表达式都必须包括左括号和右括号。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
- 向引用数组元素的表达式添加右括号。
