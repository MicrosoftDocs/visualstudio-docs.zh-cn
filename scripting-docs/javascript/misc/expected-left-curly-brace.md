# <a name="expected-"></a>应有“{”
未包括标记函数体开始的左大括号。 构成函数主体的代码（即使是单个行）必须包含在左大括号和右大括号中。 请注意，在函数主体中使用大括号比在循环中使用更严格。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
- 添加标记函数体开头的左大括号。
