# <a name="expected-var"></a>应有“var”
应为 "var" (ERRnoVar 1034) )   
  
 无法在需要时添加关键字 var。  
  
 有关 var 关键字的详细信息，请参阅 [Var 语句](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Statements/var)。
