# <a name="expected--javascript"></a>应有“=”(JavaScript)
您尝试创建一个与条件编译语句一起使用的变量，但没有在变量和要分配给它的值之间放置等号。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
- 添加等号。 例如：  
  
    ```JavaScript  
    @set @myvar1 = 1  
    ```
