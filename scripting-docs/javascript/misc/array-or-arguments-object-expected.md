# <a name="array-or-arguments-object-expected"></a>缺少 Array 或 arguments 对象
未将数组作为参数提供。 此错误仅适用于**函数。** 如果指定此参数，则此函数的第二个参数必须是 `Array` 对象或 **参数** 对象。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
- 将 `Array` 或 **参数** 对象指定为第二个参数。
