# <a name="expected-"></a>应用“;”
您尝试在一行上放置多个语句，但没有用分号分隔。 分号用于终止语句。 虽然您可以将多个语句放置在单个行上，但每个语句必须使用分号进行分隔。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
- 用分号标记每个语句的末尾。  
  
- 请确保函数调用正确地使用括号。  
  
- 请确保未忘记 **for** 循环标头内的分号。
