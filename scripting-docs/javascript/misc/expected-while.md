# <a name="expected-while"></a>应有“while”
未 `while` 在循环中包含条件 `do...while` 。 **Do**语句必须在代码块的末尾有一个相应的 `while` 测试。  
  
### <a name="to-correct-this-error"></a>更正此错误  
  
- 在 `while` 右大括号后面包含测试语句。
