# <a name="how-to-change-fonts"></a>如何：更改字体
可更改 Spy++ 窗口的字体、字体样式和字体大小。

### <a name="to-change-font-options"></a>更改字体选项

1. 从“视图”菜单中，选择“字体”。

2. 选择字体、字形和字号。

3. 单击 **“确定”** 。

   选择“保存为默认字”将导致之后所有的 Spy++ 窗口均使用此字体。
