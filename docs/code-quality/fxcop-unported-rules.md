# <a name="unported-rules"></a>已取消移植的规则

## <a name="overview"></a>概述

尚未作为 Roslyn 分析器实现的 FxCop 传统分析规则集包含以下两个存储桶：
- 尚未移植但仍 [可移植](fxcop-unported-rules-may-get-ported.md)的规则。
- 不推荐使用且 [不会移植](fxcop-unported-deprecated-rules.md)的规则。
