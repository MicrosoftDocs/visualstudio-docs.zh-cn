# <a name="multiple-categories-are-present"></a>出现多个类别
将选中在给定时间范围内包含多个活动类型的部分。 请注意，将会显示两种颜色，并且一种颜色垂直堆叠在另一种颜色之上。 这两种颜色表示给定时间范围内最常见的两种活动。 这两种活动出现的频率在竖条上以 2:1 或 1:1 的比例显示。 若要查看更多详细信息，请尝试放大线程视图的这一部分。
