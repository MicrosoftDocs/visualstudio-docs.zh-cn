# <a name="how-to-programmatically-group-rows-in-a-worksheet"></a>如何：以编程方式对工作表中的行进行分组
  您可以对一个或多个整行进行分组。 若要在工作表中创建组，请使用 <xref:Microsoft.Office.Tools.Excel.NamedRange> 控件或本机 Excel 范围对象。
