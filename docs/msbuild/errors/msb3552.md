# <a name="msb3552-resource-file-filename-cannot-be-found"></a>MSB3552：找不到资源文件“filename”

当找不到请求资源文件时，会发生此错误。 检查文件名的路径和拼写。 这些文件通常指定为项目文件中的 `EmbeddedResource` 项。
