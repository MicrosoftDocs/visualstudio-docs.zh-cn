# <a name="idebugport2getportid"></a>IDebugPort2::GetPortId
获取端口标识符。

## <a name="syntax"></a>语法

```cpp
HRESULT GetPortId( 
   GUID* pguidPort
);
```

```csharp
int GetPortId( 
   out Guid pguidPort
);
```

## <a name="parameters"></a>parameters
`pguidPort`\
弄返回用于标识端口的 GUID。

## <a name="return-value"></a>返回值
 如果成功， `S_OK` 则返回; 否则返回错误代码。
