# <a name="idebugstackframe2getname"></a>IDebugStackFrame2::GetName
获取堆栈帧的名称。

## <a name="syntax"></a>语法

```cpp
HRESULT GetName ( 
   BSTR* pbstrName
);
```

```csharp
int GetName ( 
   out string pbstrName
);
```

## <a name="parameters"></a>parameters
`pbstrName`\
弄返回堆栈帧的名称。

## <a name="return-value"></a>返回值
 如果成功， `S_OK` 则返回; 否则返回错误代码。

## <a name="remarks"></a>备注
 堆栈帧的名称通常是正在执行的方法的名称。
